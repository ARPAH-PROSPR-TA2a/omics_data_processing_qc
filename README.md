# omics_data_processing_qc
Code used to generate omics assay QC information

Notes:

* Each study has folder named after it (i.e. VIBRANT, DPP, DEFINE-HF, PRESERVED-HF, CALERIE, SELECT).

* Each study folder contains a subdirectory for each omic (i.e. DNAm, proteomics, metabolomics).

* In cases where there are >1 array type (Somalogic and Olink proteomics), there will be a subdirectory for each one (i.e. proteomics/somalogic, proteomics/olink).
